# Contributing to Neurix UI

Nous apprécions votre contribution ! Ce document fournit les lignes directrices pour contribuer au projet.

## 🤝 Comment contribuer

### Rapporter des bugs
- Vérifiez que le bug n'a pas déjà été signalé
- Fournissez une description claire et concise
- Donnez des étapes pour reproduire le problème
- Annexez les captures d'écran ou logs si pertinent

### Suggérer des améliorations
- Utilisez un titre clair et descriptif
- Fournissez une description détaillée de la suggestion
- Listez les exemples ou cas d'usage

### Pull Requests
1. Fork le projet
2. Créez une branche (`git checkout -b feature/AmazingFeature`)
3. Committez vos changements (`git commit -m 'Add AmazingFeature'`)
4. Poussez vers votre fork (`git push origin feature/AmazingFeature`)
5. Ouvrez une Pull Request

## 📋 Code Style

- Utilisez 2 espaces pour l'indentation
- Respectez la structure existante du projet
- Écrivez du code lisible et commenté
- Testez votre code avant de soumettre

## 📦 Setup de développement

```bash
# Installer les dépendances
npm install

# Lancer le serveur de dev
npm run dev

# Build pour production
npm run build
```

## ✅ Checks avant de soumettre

- Code propre et formaté
- Pas d'erreurs de console
- Tests passants
- Documentation à jour

Merci pour votre contribution ! 🎉
